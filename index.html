Projects

VGA Controller

Designed and implemented a Video Graphics Array (VGA) controller on the DE1-SoC board. The system achieved a 1280x1024 resolution at 60 Hz refresh rate. Key features include signal synchronization using a phase-locked loop (PLL), graphical figure manipulation, and color control via switches. This project demonstrated proficiency in VGA signal generation, synchronization logic, and hardware design using VHDL.

Digital Audio System

Developed a digital audio system that captures analog sound via a microphone, converts it to digital signals, processes the data, and plays it back through speakers. Implemented with the DE1-SoC board and the WM8731 CODEC, the system utilized real-time audio processing and achieved high-quality audio output with minimal latency.

Learn More

HPS-FPGA Integration

Integrated the Hard Processor System (HPS) and FPGA sides of the DE1-SoC board to process accelerometer data and display tilt angles on LEDs in real-time. The project utilized memory-mapped I/O for efficient hardware-software communication, showcasing seamless integration of embedded systems concepts.

Learn More

RISC CPU Design

Currently designing a custom RISC CPU architecture with a focus on optimizing instruction sets and control logic. This project involves implementing core components such as the ALU, control unit, and memory access using VHDL. The CPU will be simulated and synthesized for functional verification.

Learn More

IR Signal Decoding

Created a system to decode NEC protocol infrared signals using the DE1-SoC board. The project involved implementing a state machine to process IR signals and display decoded data on seven-segment displays, enabling remote-control functionality.
